There are three list types in HTML:

  • unordered list — used to group a set of related items in no particular order.
  • ordered list — used to group a set of related items in a specific order.
  • description list — used to display name/value pairs such as terms and definitions.
Unorder list - This will list items using plain bullets.
An unordered list is a collection of related items that have no special order or sequence. Each item in the list is marked with a bullet(default). 
Attribute : < tag type="_value">< / tag >
CSS : style="list-style-type:_value";
  • disc Sets the list item marker to a bullet
  • circle Sets the list item marker to a circle or with css style="list-style-type: disc";
  • square Sets the list item marker to a square
  • none The list items will not be marked
Order list - List displays elements in numbered format.
We can use ordered list to represent items either in numerical order format or alphabetical order format,
  1. type="1" The list items will be numbered with numbers (default)
  2. type="A" The list items will be numbered with uppercase letters
  3. type="a" The list items will be numbered with lowercase letters
  4. type="I" The list items will be numbered with uppercase roman numbers
  5. type="i" The list items will be numbered with lowercase roman numbers
Description list - is a list of items with a description of each item term.
<dl> tag
defines the description list.
<dt> tag
defines data term.
<dd> tag
defines data description.
